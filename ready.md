## install python3
  1. [官网](https://www.python.org/downloads/)下载特定系统的安装包 或者 brew install python3  
  2. cmd 输入 python3 启动  
  3. pip3 搭配 python3 使用  

## install package
  1. pip3 install numpy  
  2. pip3 install scipy  
  3. pip3 install matplotlib  
  4. pip3 install scikit-learn
